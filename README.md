# AstroFit Backend

Backend for AstroFit, providing APIs and database operations for the application.

## Features

- User authentication
- Database integration
- REST API endpoints
- Fitness data management

## Tech Stack

- Python
- Flask
- SQL Database

## Setup

```bash
git clone https://github.com/sinxcos07/astrofit-backend.git
cd astrofit-backend

pip install -r requirements.txt
python app.py
```
